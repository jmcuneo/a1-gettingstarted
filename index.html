Click and drag the ball to move it. Let go of the ball while it's still moving to let it move on its own!

Radius

The width of the ball. Minimum value is 5, maximum value is 100.

Friction

How much the ball's horizontal speed gets reduced by while touching the ground. Minimum value is 0, maximum value is 5.

Gravity

How much the ball's vertical speed gets increased by while not touching the ground. Negative values result in the ball falling upwards, and a value of 0 results in the ball moving at constant speed unless coliding with a wall. Minimum value is -5, maximum value is 5.

Max Velocity

The maximum speed that the ball is allowed to move at along either axis. A value of 0 prevents the ball from moving. Minimum value is 0, maximum value is 100.

Bounciness

The amount of speed that the ball retains when hitting a wall. Values below 1 result in the ball losing speed, and above 1 result in the ball gaining speed. Minimum value is 0, maximum value is 2.
